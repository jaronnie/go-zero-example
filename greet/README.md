# greet

## 入门使用

使用 goctl 工具创建 greet 服务

```shell
cd github.com/gocloudcoder/go-zero-example
goctl api new greet
```

安装go-zero以及解决依赖问题

```shell
cd greet
GO111MODULE=on GOPROXY=https://goproxy.cn/,direct go get -u github.com/tal-tech/go-zero
go mod tidy
```

运行程序

```shell
go run greet.go -f etc/greet-api.yaml
```

访问接口

```shell
curl localhost:8888/from/you
```

修改handler/greetHandler.go

```go
package handler

import (
	"net/http"

	"greet/internal/logic"
	"greet/internal/svc"
	"greet/internal/types"

	"github.com/tal-tech/go-zero/rest/httpx"
)

func QueryPasswordHandler(ctx *svc.ServiceContext) http.HandlerFunc {
	return func(w http.ResponseWriter, r *http.Request) {
		var req types.Request
		if err := httpx.Parse(r, &req); err != nil {
			httpx.Error(w, err)
			return
		}

		l := logic.NewGreetLogic(r.Context(), ctx)
		resp, err := l.QueryPasswordByUsername(req)
		if err != nil {
			httpx.Error(w, err)
		} else {
			httpx.OkJson(w, resp)
		}
	}
}
```
